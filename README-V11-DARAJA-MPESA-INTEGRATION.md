# Shule AI V11 Daraja / M-PESA Integration

This frontend package adds the approved payment/subscription UI templates and wires them to backend Daraja STK Push endpoints.

## Added
- Parent Payments & Subscriptions dashboard
- Parent Pay School Fees modal
- Parent Shule AI Subscription modal
- Admin School Payment Details section
- Super Admin Platform Payment System section
- Name-change request payment prompt flow
- Sidebar entries for Admin Payment Details and Super Admin Payment System
