# Shule AI Frontend V9.2

## Added

### Admin Departments UI
Admin Dashboard now has a **Departments** sidebar item.

Admin can:
- Create department
- Select department head
- Assign multiple teachers
- Edit department
- Archive department
- Open teacher group chat

When a department is created, backend creates its group chat. It appears in the teacher **Messages → Groups** tab together with **Staff Room**.

### Global Profile Pictures
Profile pictures are now global:
- Header avatar
- Sidebar/profile areas
- Student/teacher cards where profile image exists
- Department teacher selector
- Any global avatar element

Profile images can be clicked to view full-size.
